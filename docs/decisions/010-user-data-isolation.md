# ADR-010 用户数据隔离

- 状态：Accepted
- 更新时间：2026-03-02

## 决策

系统按用户维度进行数据隔离，任何项目、文件、任务、对话查询都必须经过用户边界校验。

## 适用范围

- 项目与文件访问
- 生成任务查询与下载
- 对话与 RAG 检索

## 当前状态

- [x] 认证体系已接入主要 API
- [x] 关键资源路由已做用户权限检查
- [ ] 需要持续补齐边缘接口一致性检查

## 实施要求

- 路由层统一使用认证依赖注入
- service/data 查询必须包含用户边界条件
- 鉴权失败返回 `401`，越权返回 `403`

## 验收标准

- [ ] 任意用户无法读取他人项目与文件
- [ ] 任意用户无法下载他人生成结果
- [ ] 自动化测试覆盖越权场景
