# Standard Docs Alias

`docs/standard` is a compatibility alias for `docs/standards`.

Please read and maintain the canonical files under:

- `docs/standards/README.md`
- `docs/standards/backend.md`
- `docs/standards/frontend.md`
- `docs/standards/documentation.md`
- `docs/standards/git.md`
- `docs/standards/AI_COLLABORATION.md`

When adding or updating standards, update `docs/standards/*` first.
