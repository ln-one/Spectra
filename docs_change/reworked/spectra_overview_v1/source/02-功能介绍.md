# 功能介绍

当前作品已经做成统一工作台，教师在一个界面里就能完成资料查看、会话生成、结果承接和后续操作。

![](../../spectra_v11/assets/media/spectra-frontend-workbench.png){width="7.0in" height="3.95in"}
图 2 前端工作台截图，展示教师如何在同一界面内查看资料、推进会话并承接多模态结果。

![](../../spectra_v11/assets/media/image3.png){width="6.8in" height="4.0in"}
图 3 `Studio` 统一内容工坊示意图，展示课堂主交付、互动练习和教学辅助如何围绕同一课程空间组织。

围绕这套工作面，系统当前可支持：

- `PPT` 课件、`Word` 教案、思维导图等多种结果生成；
- 互动小测、互动游戏、学情预演等课堂辅助内容；
- 预览、修改、历史查看与下载等结果操作。
