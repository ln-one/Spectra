# 7.3 系统效果展示

# 7.3.1 用户前端界面展示

# 1、首页

在用户登入该产品域名后，首先会进入首页看到不同城市的图片展示和相应的切换动画效果，用户可以点击该界面右上角的‘汉堡包’图标进入关于界面或者点击该界面中部的‘开始旅行’按钮进入系统登陆界面。

![](images/8a2662b164a7321231a83cc1f09053c31db80efd22904d40f9ae201865b8928a.jpg)
图7-1首页界面展示

# 2、关于界面

用户进入关于界面，该界面可以为用户推荐一些旅游目的地、旅游期刊、团队介绍等。

说明：

(1) 该窗体左侧可以选择旅行目的地，用户可以对其想要前往的城市进行选择，这有利于模型更好地对前往的城市进行信息的采集和回答的反馈。
(2) 该窗体右侧 travel tips 可以给用户们对于不同旅行需求提供旅行建议等。
(3) 该窗体右侧 Resources 将提供一些较为出名的旅行杂志，用户点击相关的内容网页将帮助用户前往相应的杂志首页。
(4) 该窗体右侧 About Us 将提供一些开发团队的具体信息，用户点击相关选项可进行获取。

![](images/623239148d2f76a92ee59965528a0e1b2a3baa02547538f0e695f5054be1420c.jpg)
图7-2关于界面展示

# 3、登录注册界面

用户想要进入系统需输入用户名和密码登录自己的账号，同时新用户可以注册自己的账号。在注册登录的过程中，系统会对用户名、电子邮箱、密码等格式进行验证。

- 总体要求如下：

(1) 用户名至少四个字符。
(2) 电子邮箱格式必须正确。
(3) 密码至少八个字符，同时需具有符号、字母或数字其中两种的排列组合。
(4) 确认密码和密码必须相同。

- 注：对于不符合该要求的内容，系统会给出错误提示，告知用户各个信息合法格式。

![](images/57c0f08d080067b76c83f047d6dc380c9b015cd8b00a4091a8f354d426a0eaa8.jpg)
图7-3注册界面展示

![](images/1f4e2c4f0f6dd4ab7bd32c4043ebfdba2002640c42c2130c2c45780dbc16f60a.jpg)
图7-4登录界面展示

# 4、语言选择

用户可以通过点击语言图标按钮，选择系统的显示语言，本系统支持五种语言切换，分别为英文、简体中文、俄语、法语、西班牙语。

在选择语言的时候请点击相应语言右边的圆框，当圆框内有黑心圆点时即表明选择成功，然后点击右上角的 save 按钮将实现产品语言设置的全局切换。

![](images/50fb7b685ea94a5c1825ebadb08a3da46f5e273d62ed9153148fc99b9b1aedd3.jpg)
图7-5显示语言选择

# 9、问题信息存储及多次使用

该系统支持用户提前编辑要提出的问题内容，并将其保存下来，以便后续直接选择进行提问，用户可点击对话框左下角“聊天”图标，进行问题信息的存储和使用功能。

使用说明：用户可将该问题的标题和提示信息填入“Title”文字框，以便后续提示自己该问题的内容，将问题的具体信息填入“Add a new prompt”文字框，这即为所要提问问题的具体内容。

两个文字框均正常填入后点击窗口下面的“ADD PROMPT”按钮，该问题标题将显示在“Title”文字框的上方，可以点击标题右边的“铅笔”图标或“垃圾桶”图标分别实现问题信息的编辑或问题的删除。

如果用户想使用保存的问题时，可以直接点击问题的标题，系统会自动将问题传入大模型并进行回答。

# 4、对话设置管理

用户可点击左侧导航栏的Chat中Settings按钮进行对话功能设置

![](images/78bfd2c13f796b1abddf41702a132746463bccc0cd0af0cccb31208990ba2807.jpg)
图7-22对话设置管理页面

下面将对上图设置进行介绍：

(1) openai api key: 用户共享的模型的 api key
(2) open api_key_settings: 决定是否允许用户进行私人 api_key 的设置。
(3) open_web_search: 决定是否允许用户进行 web_search 模式提问。
(4) open-registration: 决定是否允许用户进行非监督注册。
(5) open_frugal_mode_control: 决定是否允许用户开启节俭模式。
