#### 3.3 教学法库

**预置教学法模型**：

1. **5E教学法**（理科实验课）
   - Engage → Explore → Explain → Elaborate → Evaluate
   - 适用：物理、化学、生物实验课

2. **BOPPPS教学法**（高效课堂）
   - Bridge → Objective → Pre-test → Participatory → Post-test → Summary
   - 适用：各学科标准课时

3. **PBL问题导向法**（探究式学习）
   - 问题提出 → 小组讨论 → 资料查找 → 方案设计 → 成果展示
   - 适用：综合实践课、项目式学习

4. **翻转课堂法**（混合式教学）
   - 课前自学 → 课中讨论 → 课后巩固
   - 适用：需要预习的课程

5. **传统讲授法**（理论课）
   - 导入 → 新授 → 练习 → 总结 → 作业
   - 适用：概念讲解、知识传授
