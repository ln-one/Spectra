## 原型制作建议

### 工具选择
1. **Figma**（推荐）
   - 在线协作
   - 组件库丰富
   - 免费版功能足够

2. **Sketch**
   - Mac专用
   - 插件生态好
   - 适合高保真设计

3. **Axure RP**
   - 交互原型强大
   - 适合复杂流程
   - 学习曲线较陡

4. **draw.io / Excalidraw**
   - 快速绘制
   - 适合低保真原型
   - 免费开源

### 制作步骤

1. **建立设计系统**
   - 定义颜色规范
   - 定义字体规范
   - 创建组件库

2. **绘制关键页面**
   - 先画主流程页面
   - 再画辅助页面
   - 最后画异常页面

3. **添加交互**
   - 定义页面跳转
   - 定义状态变化
   - 定义动画效果

4. **用户测试**
   - 邀请教师试用
   - 收集反馈意见
   - 迭代优化

## 设计规范参考

### 颜色方案
- 主色：#4A90E2（蓝色，专业、可信）
- 辅助色：#7ED321（绿色，成功、积极）
- 警告色：#F5A623（橙色，警告）
- 错误色：#D0021B（红色，错误）
- 中性色：#4A4A4A、#9B9B9B、#F5F5F5

### 字体规范
- 标题：思源黑体 / PingFang SC
- 正文：苹方 / Microsoft YaHei
- 代码：Consolas / Monaco

### 间距规范
- 基础单位：8px
- 常用间距：8px、16px、24px、32px、48px

### 圆角规范
- 按钮：4px
- 卡片：8px
- 弹窗：12px
