## 使用建议

- 首次喂外部系统时，答辩场景优先使用 `答辩版`。
- 写正文、做方案书、希望展示更强结构压强时，使用 `文档版本`。
- 单版本图默认按文件内注明的场景投喂。
- 所有主 Prompt 默认使用中文，英文只作为补充关键词。
