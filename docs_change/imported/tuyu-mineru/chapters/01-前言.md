# 1.4术语及缩略语

途语文旅推荐系统（CulturaPath Recommendation System）：途语文旅推荐系统是一个综合性的平台，用于处理和回答有关旅游相关的查询。它结合了大数据处理、知识库支持、用户交互对话以及后台管理功能，旨在为用户提供详尽、精确的文旅信息和建议。通过自动爬取、数据处理、向量化搜索和对话管理，系统能够理解用户的意图，并提供个性化的旅行建议和信息。

robots.txt：一种用于网站根目录下的文本文件，它通过“爬虫排除标准”（Robots Exclusion Protocol, REP）告诉网络爬虫（也称为机器人或搜索引擎爬虫）哪些页面可以被抓取，哪些不应该被抓取。这个标准并不是由任何正式的互联网标准组织制定的，但它已被广泛接受和实施，成为了搜索引擎索引网站内容时的一种自律行为准则。

selenium 自动化 和 WebDriver: Selenium 通过使用 WebDriver 支持市场上所有主流浏览器的自动化。Webdriver 是一个 API 和协议，它定义了一个语言中立的接口，用于控制 web 浏览器的行为。每个浏览器都有一个特定的 WebDriver 实现，称为驱动程序。驱动程序是负责委派给浏览器的组件，并处理与 Selenium 和浏览器之间的通信。

XPath：XPath表示XML路径语言。它使用非XML语法来提供一种灵活地定位（指向）XML文档的不同部分的方法。它也可以用于检查文档中某个定位节点是否与某个模式（pattern）匹配。XPath使用路径标识符（类似于URL）通过层级结构来导航XML文档。它使用非XML语法，使得它可被用于URI和XML属性值。

CSS 选择器：CSS（层叠样式表）用于设置和布置网页——例如，更改内容的字体、颜色、大小和间距，将其拆分为多个列，或添加动画和其他装饰功能。

正则表达式：正则表达式是一种文本模式，包括普通字符（例如，a 到 z 之间的字母）和特殊字符（称为"元字符")，可以用来描述和匹配字符串的特定模式。正则表达式是一种用于模式匹配和搜索文本的工具。正则表达式提供了一种灵活且强大的方式

来查找、替换、验证和提取文本数据。

图片的Base64编码：是网络上最常见的用于传输8Bit字节码的编码方式之一，Base64就是一种基于64个可打印字符来表示二进制数据的方法。可查看RFC2045～RFC2049，上面有MIME的详细规范。Base64，就是包括小写字母a-z、大写字母A-Z、数字0-9、符号“+”、“/”一共64个字符的字符集，（任何符号都可以转换成这个字符集中的字符，这个转换过程就叫做base64编码。图片的base64编码就是可以将一幅图片数据编码成一串字符串，使用该字符串代替图片地址，从而不需要使用图片的URL地址。

结构化地址：首先，地址肯定是一串字符，内含国家、省份、城市、区县、城镇、乡村、街道、门牌号码、屋邨、大厦等建筑物名称。按照由大区域名称到小区域名称组合在一起的字符。一个有效的地址应该是独一无二的。

RAG：检索增强生成（Retrieval Augmented Generation, RAG），它通过从数据源中检索信息来辅助大语言模型（Large Language Model, LLM）生成答案。

LLM：大语言模型（Large Language Model，LLM），也称大型语言模型，是一种人工智能模型，旨在理解和生成人类语言。它们在大量的文本数据上进行训练，可以执行广泛的任务，包括文本总结、翻译、情感分析等等。LLM的特点是规模庞大，包含数十亿的参数，帮助它们学习语言数据中的复杂模式。

LoRA 微调：LoRA 微调一种提升大模型在下游任务性能的有效方法，它通过固定原始模型参数，仅训练低秩矩阵的分解部分，实现高效且精确的模型调整。

基于 HNSW 的向量索引：HNSW 是一种基于图的近邻搜索算法，是对 NSW 的进一步优化，利用指数衰减概率函数构建多层图，提高了搜索效率和准确性。

基于L2的向量距离计算方式：向量的L2范数是指一个多维向量在坐标系中的距离，也叫Euclidean范数，简称L2范数。

Nginx: Nginx 是一种高性能的 HTTP 和反向代理服务器，也用作邮件代理服务器。它是一个多用途的服务器，能够处理大量的并发连接，通常用于提高网站的性能和稳定性。

SSL（Secure Sockets Layer）：安全套接层，是一种安全协议，用于在互联网上的计算机网络之间提供加密通信。SSL 通过在传输数据前对其进行加密，确保数据传输过程中的安全和隐私。

域名：域名是互联网上某个网站的唯一地址，用于在网络上定位和识别计算机。域名系统（DNS）将易于记忆的域名转换为计算机用于寻址的数字IP地址。

Nuxt.js：Nuxt.js 是一个基于 Vue.js 的开源框架，用于创建服务端渲染（SSR）或静态生成的Vue.js 应用程序。它为客户端和服务器端代码提供了一个统一的编程模型，并简化了开发过程。

Django: Django 是一个高级的 Python Web 框架，它鼓励快速开发和干净、实用的

设计。Django遵循模型-视图-模板（MVT）架构模式，适用于快速开发复杂的、数据库驱动的网站。

Vue.js: Vue.js 是一个渐进式 JavaScript 框架，用于构建用户界面。它易于集成到其他项目和库中。Vue 的核心库只关注视图层，易于学习，适合用于创建单页应用。

IP地址：网络中每个设备的唯一标识符。它是一串数字，用于在互联网上识别每台计算机或设备。

前后端：前端通常指的是用户直接与之交互的应用程序部分，如网页或应用程序界面，而后端是服务器端的组成部分，负责处理前端发送的请求、数据处理和存储。

中间件：中间件是一种软件服务，用于连接两个应用程序或两个组件之间的数据流。它使得不同的技术可以通过一个共同的接口进行通信和数据交换。

组件：在软件和Web开发中，组件是可重用的代码块，用于构建用户界面。它们封装了自己的结构、样式和行为，并可以在不同的部分和项目中重复使用。

样式：指定应用程序或网页视觉外观的指令和指南。通常通过 CSS（层叠样式表）来实现，定义了字体、颜色、布局等。

SSR（服务器端渲染）：服务器端渲染是指在服务器上生成完整的页面，然后发送到客户端的过程。这种方法可以提高首次加载速度，并优化搜索引擎优化（SEO）。
