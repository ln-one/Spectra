## 界面布局设计

### 主界面布局

**三栏布局**：

```
┌─────────────────────────────────────────────────────────┐
│  [Logo] Spectra              [用户] [设置] [帮助]        │
├──────────┬──────────────────────────────────┬───────────┤
│  侧边栏  │  主工作区                         │  信息面板 │
│          │                                  │           │
│  项目列表│  对话区 / 预览区                  │  参考资料 │
│  ┌────┐  │                                  │  知识库   │
│  │项目1│  │                                  │  提示     │
│  ├────┤  │                                  │           │
│  │项目2│  │                                  │           │
│  ├────┤  │                                  │           │
│  │项目3│  │                                  │           │
│  └────┘  │                                  │           │
│          │                                  │           │
│  [新建]  │                                  │           │
└──────────┴──────────────────────────────────┴───────────┘
```

### 响应式设计

**桌面端（>1200px）**：

- 三栏布局
- 侧边栏固定
- 主工作区和信息面板可调整宽度

**平板端（768px-1200px）**：

- 两栏布局
- 侧边栏可折叠
- 信息面板变为抽屉式

**移动端（<768px）**：

- 单栏布局
- 底部导航栏
- 全屏对话/预览

## 交互状态设计

### 按钮状态

- **默认**：正常显示
- **悬停**：颜色加深，显示提示
- **点击**：按下效果
- **禁用**：灰色，不可点击
- **加载**：显示加载动画

### 输入框状态

- **默认**：灰色边框
- **聚焦**：蓝色边框，显示光标
- **错误**：红色边框，显示错误信息
- **成功**：绿色边框，显示成功图标

### 对话状态

- **用户消息**：右对齐，蓝色背景
- **AI消息**：左对齐，白色背景
- **系统消息**：居中，灰色背景
- **加载中**：显示"正在思考..."动画

## 错误处理

### 常见错误场景

1. **网络错误**
   - 提示："网络连接失败，请检查网络后重试"
   - 操作：[重试] [取消]

2. **文件上传失败**
   - 提示："文件上传失败，可能是文件过大或格式不支持"
   - 操作：[重新上传] [查看支持格式]

3. **语音识别失败**
   - 提示："未识别到内容，请重试或使用文字输入"
   - 操作：[重新录音] [文字输入]

4. **生成失败**
   - 提示："课件生成失败，请稍后重试或联系客服"
   - 操作：[重试] [保存草稿] [联系客服]

5. **权限错误**
   - 提示："需要麦克风权限才能使用语音输入"
   - 操作：[开启权限] [使用文字输入]
