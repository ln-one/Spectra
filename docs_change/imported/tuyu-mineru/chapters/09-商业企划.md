# 商业企划

# 9.1基于PEST的市场分析

# 9.1.1政策环境（P）

文旅产业垂直应用的大语言模型，是文旅产业新兴的“智慧旅游”领域，在近年来受到国家的高度重视。为推动AI技术的落地应用，促进行业的高新发展，政府颁布了一系列针对性强、操作性高的政策，旨在推动文旅产业的数字化转型和创新发展。国务院在2021年12月22日发布了《“十四五”旅游业发展规划》，明确提出利用大数据、人工智能等先进技术提升旅游服务质量和游客体验，强调通过互联网有效整合线上线下资源，推进以数字化、网络化、智能化为特征的智慧旅游，深化“互联网+旅游”，扩大新技术场景应用，文旅产业大语言模型作为重要的技术支撑，被寄予厚望。

在鼓励创新的同时，政府也加强了对文旅产业大语言模型的监管。部分地区出台政策文件，设立专项资金支持推广“智慧旅游”，给予企业和项目一定的税收优惠和减免政策，促进文旅产业大语言模型的发展。《数据安全法》、《个人信息保护法》等相关法律法规，对文旅产业大语言模型在开发和应用过程中严格遵守数据保护和隐私法律提出了要求，确保用户数据的安全和隐私。

# 9.1.2经济环境（E）

近年来中国经济飞速发展，2023年中国GDP超126万亿元，同比增长 $5.2\%$ 。随着中国经济的高速发展，中国居民人均可支配收入也不断提升，人们已不仅限于对物质的需求，更加注重生活质量的追求，高效便捷且幸福的生活正成为人民日益增长的美好生活需要的重要组成部分，而旅游出行作为居民消费的重要品种，能够极大地提升人民生活质量，这也决定了智慧旅游产业必将迎来更大的发展空间。

文旅行业对于新技术应用的接纳度天然就很高，并且已有不少目的地、行业组织和企业通过数字化在降本、提效、保质、增收这些方面有所获益，取得了实质性的好处。而以大模型为代表的AI技术，将为文旅行业提供全新的思考逻辑框架和解决问题的方法。该行业持续需求火热，资本利好，发展长期向好。下游行业交易规模增长，为智慧文旅的相关行业提供新的发展动力。

# 9.1.3 社会环境 (S)

随着全球化的推进，社会环境呈现出文化多元与交融的特点，不同文化间的交流日益频繁，这为文旅产业提供了丰富的资源。AI对话以其独特的智能交互方式，深度融入文化传承与弘扬，能够深入挖掘并展示各地的文化特色，推动文化旅游的发展。同时，现代社会的消费者越来越注重个性化和体验，大语言模型通过智能分析消费者的兴趣和需求，提供定制化的旅游体验，满足了消费者对于独特、深度的旅游体验的追求。

从社会结构与人口特征的角度来看，文旅产业面临着多样化的受众群体。不同年龄段、不同社会阶层、不同收入水平的消费者对于旅游的需求和期望存在差异。这种智能化的服务方式不仅提高了旅游服务的效率和质量，也增强了游客的满意度和忠诚度。此外，AI对话还通过大数据分析，为文旅产业的可持续发展提供了有力支持，为文旅产业提供更为广泛的市场基础。

# 9.1.4技术环境（T）

科技进步推动自然语言处理技术、大数据和云计算技术的融合、深度学习和机器学习算法的突破和进步，基于大模型的AI对话系统具有强大的数据支持和处理能力，通过收集和分析大量旅游数据，更准确地理解游客的语言，实现高效、智能的交互，以实现文旅产业的智能化、个性化、精准化，满足游客多样化的需求，提升游客满意度。同时，具备不断学习和优化能力的系统，不断改进自身的回答和服务质量，无论是旅游信息查询、行程规划，还是景点推荐、导游服务，AI对话都能为游客提供便捷、智能的解决方案。

# 9.1.5 目标市场基本情况

# 1、未来市场的发展趋势

随着科技的不断进步和人们对智能化、个性化服务的需求日益增加，基于大模型文旅产业AI对话系统市场潜力巨大，具有广阔的发展前景。随着自然语言处理、深度学习等技术的不断完善，文旅产业大语言模型将更加精准地理解游客需求，提供更加智能化、人性化的服务；随着全球旅游业的快速发展和人们对高质量旅游体验的追求，该产业的市场规模将持续扩大，大语言模型在提升服务质量和效率方面的优势非常明显。此外，随着技术的普及和成本的降低，文旅产业大语言模型的应用范围也将进一步扩大，覆盖更多的旅游场景和服务领域。

# 2、目标客户的购买力

文旅产业大语言模型的目标客户主要包括旅游机构、在线旅游平台、旅游服务提供商以及个体旅游者。这些客户群体的购买力因地区、年龄、收入水平和消费习惯等因素而异。整体经济高速发展，随着市场规模扩大，目标客户可能会因为日益增长的美好生活需要，具有较高的消费能力和购买力，愿意为智慧文旅的智能化、个性化服务买单。

# 9.2 SWOT分析

# 9.2.1 优势 (S)

作为国内首批以人工智能技术创新为核心的高新技术企业，万维艾斯在技术和市场上具有先行者优势，易于获得市场认可。其次，公司依托南京大学人工智能推理与学习团队，拥有强大的技术研发能力和科研背景，具有国际技术转移和国际合作的经验，能够引进国际先进技术和资源，能够快速迭代和优化AI对话系统，提升产品的竞争力。另外，公司是一个产学研一体化的创新平台，集研发、技术转移、国际合作于一体，有利于打造综合性的文旅产业AI对话系统，满足不同客户的需求。

# 9.2.2 劣势（W）

在公司研发和技术创新过程中，高度依赖专业的技术研发人才，人才流动可能会对项目进度和质量构成影响。虽然目前公司在技术研发方面有优势，但作为技术导向型公司，市场推广经验有限、对文旅行业需求理解可能不足，可能在市场推广和品牌建设方面相对薄弱，缺乏对文化旅游行业特定需求和趋势的深入理解。同时，知识产权保护体系不完备。目前我国知识产权保护由于在法律和管理体制等方面不够完善，存在着多头管理，侵权打击不力，侵权成本低。导致现阶段我国知识产权整体保护意识淡薄，保护水平低下，并且维权成本高，这是公司可能难以承受的。

# 9.2.3 机会（0）

具有时机优势。随着文旅产业数字化转型的深入，文旅产业数字化转型需求增加，对AI对话系统等智能化服务的需求日益增长。国家和地方政府对人工智能产业给予重点扶持，有助于公司获取资金和政策支持。关于产品应用，可以与文旅产业内的景区、旅游平台、文化传媒等企业建立跨界合作，拓展业务应用场景和市场份额；通过国际合作，有机会将业务扩展到海外市场，把握国际旅游恢复增长的机会。

# 9.2.4威胁（T）

公司本身具备的竞争优势在日益增长的市场竞争中存在不小的威胁因素。随着越来越多企业和创新技术涌入文旅AI领域，市场竞争将日益激烈，不具备过硬的竞争实力和品牌能力，难以与市场中其他大型企业抗衡。目前正是AI对话产品的新潮热潮时期，技术更新速度快，迭代速度快，需要持续投入研发资源以维持技术优势。同时，国内外对AI技术特别是人工智能应用的法律法规日趋完善，可能会带来新的合规要求和限制，在处理用户数据时，面临数据安全和隐私保护的挑战，任何泄露都可能严重影响公司声誉。

# 9.2.5 企业的核心竞争优势

1、拥有强大的技术研发背景和团队优势。依托南京大学的人工智能推理与学习团队，该公司聚集了一批高水平的科研人才和技术专家，确保了其在人工智能技术创新方面的领先地位。这一点为公司在开发高度复杂和高度定制化的AI对话系统提供了坚实的基础，使其能够快速响应市场变化，持续推出技术领先的产品和服务。
2、产学研一体化模式是其另一大核心竞争优势。通过紧密的产学研合作，万维艾斯不仅能够及时将最新的科研成果转化为实际的产品和技术，还能够吸引和培养更多的人才，进一步增强其在人工智能领域的创新能力和技术积累。
3、万维艾斯的国际化视野和合作经验也是其重要的竞争优势之一。通过国际技术转移和合作，万维艾斯不仅能够引进国际先进技术，还能够将自身的技术和产品推向国际市场，这为公司拓展海外业务和形成全球竞争力提供了可能。
4、作为国内首批以人工智能技术创新为核心的高新技术企业，万维艾斯在行业中享有较高的知名度和品牌影响力，这有助于其在与文旅产业相关企业建立合作、拓展市场时获得信任和支持。

# 9.3管理团队

# 9.3.1 管理理念

我们团队旨在培养一群善于沟通、理解、解决问题的人。首先，我们确保建立一个积极且相互尊重的工作关系，良好且积极的工作关系可以大大提高员工的积极性；我们应当关注员工们思考和讲话的方式并适应他们的文化、语言，尝试融入员工，尽量不产生交流和思想上的代沟。其次，我们力争营造“勇于承担”的团队文化。团队文化会影响每一个团队成员的行为方式。作为管理者，我们希望通过平时注意营造“勇于承担”的团队文化，让员工接受这种团队文化，那么在给员工安排工作时，团队文化就会起作用，团队成员接受工作安排的接受度就会更高。同时，我们注重前期的铺垫与沟通。对于重要的工作任务，如果搞“突然袭击”，这样很可能让员工一时很难接受，从而产生抵触情绪，导致员工都不愿意接受。

所以，对于重要工作我们会提前在团队中铺垫，让员工有一定的心理准备，这样在安排工作任务时，员工的接受度就会更高。最后，我们希望充分沟通任务的意义和资源。当与团队讨论重要工作任务的安排时，我们会选择跟团队成员充分沟通任务的意义和资源，而且会注意这两点先后的沟通顺序：先沟通任务的意义，再沟通完成任务的资源。
