# 项目综述

# 2.1 项目背景及意义

随着人工智能技术的迅速发展，自然语言处理（NLP）领域取得了显著的进步，通用型大型语言模型（LLM）已经在许多任务上取得了令人瞩目的成果。一些开源的大模型普遍是在百科、公共网络语料、开源代码库等数据上训练过的，知识分布虽然很全面，但是在一个与公众网络差距较大的领域，这类语言模型并没有包含足够的领域知识。在许多特定的垂直业务领域中，由于其与通用领域之间存在较大差异，直接采用开源的通用型LLM经常无法满足该领域应用的要求。

旅游行业是一个涉及广泛、信息繁杂的领域，包括景点介绍、旅游规划、酒店预订、餐饮推荐等多个方面，此领域类数据是一个相对容易获取且具备较高的数据价值。本项目制作专用于旅游的垂直大模型，该模型能够针对旅游领域的特点进行深度优化，通过大数据分析、自然语言处理等技术手段，实现对旅游资源的优化配置和旅游服务的智能化升级，围绕某一城市获取相关旅游数据并提供一个便捷的人机交互方式，提升用户体验、促进多模态交互技术的发展，为旅游行业提供更加智能、高效的服务，对于文旅产业的发展以及实现城市智能旅游助手等方面有着重要的意义。

# 2.2项目目标

贯彻落实科技部等六部门联合印发的《新一代人工智能发展规划》、《关于加快场景创新以人工智能高水平应用促进经济高质量发展的指导意见》等文件精神，创新建设基于旅游垂直大模型的语料库问答系统，提升大模型文本生成、语义理解和用户交互等方面的能力，为旅游行业提供更加智能、高效的服务。

通过网络、书籍等方法整理收集目标城市的文旅数据，包括但不限于城市的历史、名人、景点、饮食特色、热门店铺等信息，构建较为全面完整的语料库；基于Vue、Nuxt框架进行用户交互界面设计，搭建可视化呈现的网站平台，实现具有多样化功能特性的前端用户界面，同时基于Python、Django技术设计管理服务端，最终实现基于大模型的检索数据、自然语言智能理解和自动生成的语料库问答系统，形成功能完备、运行流畅的智能化人机交互平台。

# 2.3项目亮点

# 1、集成的大数据处理模块：

系统具有强大的数据处理能力，能自动爬取、处理并存储旅游相关数据，如美食、酒店、娱乐和景点信息，确保提供的旅行建议具有时效性和准确性。

# 2、动态的知识库支持：

系统拥有一个动态更新的知识库，支持多种数据格式，并允许用户进行数据的增删改查，保证了信息的实时更新和高度的可定制性。

# 3、智能的对话交互：

系统通过高级对话管理模块，可以理解用户意图，提供多种对话模式，确保用户可以以最便捷的方式获取所需信息。

# 4、高度定制的用户界面：

系统提供了前端界面模块，提供了丰富的定制选项，如不同城市的选择、主题模式切换（明亮模式和暗色模式）、语言设置等，增强了用户体验。

# 5、先进的LLM后端模块：

系统采用大模型技术，结合对话模型、向量模型和重排模型，使系统能够准确理解用户的提问，并提供精确的回答。

# 6、强大的后台管理功能：

系统提供了全面的后台管理模块，包括账号信息管理、用户权限管理、对话消息管理等，便于系统管理员维护和管理整个平台。

# 7、丰富的辅助功能：

系统提供了天气和新闻卡片，提供了额外的旅行信息，帮助用户更好地规划旅行。
