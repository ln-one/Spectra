## 5. 叙事主线检查

文档主线应始终保持一致：

1. 教师备课存在高频、重复、割裂的问题。
2. `Spectra` 通过多轮对话 + 多模态资料 + RAG + 生成闭环解决该问题。
3. 核心创新不仅是生成课件，更是沉淀并分享课程资料库。
4. 该创新使用户从教师扩展到学生，并进一步形成可引用、可协作、可演化的空间模型。
