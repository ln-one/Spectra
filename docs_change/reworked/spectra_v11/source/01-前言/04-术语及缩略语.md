## 术语及缩略语

| 术语 | 本文中的定义 |
| --- | --- |
| `Project` | 项目资料与结果统一管理空间，用于承接长期资料、结果和引用关系 |
| `Session` | 一次具体工作过程或会话流程，用于承接生成、修改和交互推进 |
| `Artifact` | 一次生成得到的结果对象，如课件、文档或其他成果 |
| `Version` | 结果保存后的版本锚点，用于记录正式状态 |
| `Reference` | 不同资料、结果或空间之间的引用关系 |
| `CandidateChange` | 进入后续修改、确认或版本演化的候选变更 |
| `Member` | 协作成员及其权限边界 |
| `Diego` | 内容生成模块，负责大纲、生成和迭代链路 |
| `Pagevra` | 预览、渲染与导出模块，负责结果查看和标准交付 |
| `Ourograph` | 结果状态、版本与引用管理模块 |
| `Dualweave` | 资料上传、解析与交付处理模块 |
| `Stratumind` | 检索增强与证据组织模块 |
| `Limora` | 用户、组织与权限边界模块 |

对象语言在文中保留，用于维持系统语义稳定。中文定义同时给出，用于保证方案书可读性。后文首次出现对象名时仍以中文含义为主，必要时补充英文对象名。
