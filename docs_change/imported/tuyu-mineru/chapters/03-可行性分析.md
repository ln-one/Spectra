# 可行性分析

# 3.1 社会可行性

随着旅游业的快速发展，游客对于旅游信息的需求日益高效化、个性化、多元化。旅游垂直语言大模型能够更好地满足用户需求，通过简洁方便的人机交互界面，就能帮助游客获得精准、高效、个性化的旅游信息，符合社会发展趋势。通过提高旅游服务的质量和效率，垂直大模型项目有助于提升旅游行业的整体形象和竞争力，推动旅游业的发展，产生积极的社会效益。

该系统严格符合道德标准，未涉及对于隐私相关内容的侵犯。能够通过人机的有效交互，提供丰富灵活的数据信息，实现文旅产业信息的智能化共享，为游客营造更精确高效的旅游体验，包括景点、美食、住宿等等。努力做到让广大群众享有更加方便高效的旅游出行服务，共享当代智能计算及通信技术的发展成果，不断增强人民群众的获得感、幸福感、安全感，为公司带来广泛社会影响力。

# 3.2政策可行性

我国对于人工智能和旅游业的发展都给予了高度的重视和支持。国家部署了多项与智能旅游业相关的国家重点研发计划和专项，2023年由文化和旅游部发布的《关于推动在线旅游市场高质量发展的意见》明确指出要推动在线旅游经营者深度应用5G、人工智能、大数据、云计算、区块链等新技术，以科技引领行业创新发展，鼓励在线旅游经营者利用人工智能等技术开展业态创新融合如智慧旅游，推动旅游经营者数字化转型升级，为旅游垂直大模型项目的发展提供了坚实的政策基础。此外，国家旅游局、国务院印发的《“十四五”旅游业发展规划》和《关于促进智慧旅游发展的指导意见》围绕文旅业转型升级提出了未来发展目标，都标志着中国的旅游产业智能化转型已经步入了快车道。将人工智能与文旅产业相结合，是我国智慧旅游相关政策的重点方向之一。

随着旅游业的迅猛发展和数字化转型的加速推进，搭建一个垂直应用于旅游产业的语言大模型已成为提升旅游服务质量和效率的关键举措。这一模型能够深度挖掘旅游领域的海量数据，精准把握游客的个性化需求和旅游市场的动态变化。国家推进智慧旅游产业变革，推广智能化、数字化、信息化技术应用，推动发展智慧旅游，为旅游行业提供强大的决策支持。

# 3.3 技术可行性

本组成员都有一定的组织编程开发项目经验，来自多个学院，专业知识扎实且全面，对本项目的完成有很大信心，具有较强的自学能力和团队协作能力。大部分同学已经拥有一定的理论知识和系统设计实践经验，能够担负技术重任。项目使用python作为语言基础，使用灵活，方便，拓展性强。深度学习的相关技术也是项目实现的有力保障。虽然项目工作量较大，一些技术开发存在技术难点，但通过开发用户的技术攻关和良好的协调安排，小组成员有能力在比赛限定时间内完成项目。因此本项目在技术上可行。

# 3.4经济可行性

旅游业市场庞大，对于智能旅游服务的需求持续增长，为垂直大模型项目提供了广阔的市场空间。完成该语料库对话模型开发的支出仅为项目开发需要的基础设备，耗材使用、购买参考书等学习资料，一些简单的硬件购买成本以及少量市场开拓、广告、运营，后期维护成本。而本项目完成度高，系统性，一体化强，对提高服务效率、降低运营成本有精确、及时的优点，可以实现成本效益的平衡；同时设计了后台管理与web页面等多种控制方式，方便使用人员进行操作。若投入到各类旅游企业中合作使用、开展增值服务等，所收获的效益远大于成本，实现盈利模式的多样化，因此经济可行。

# 3.5 实践可行性

小组有两名指导教师，李老师为控制方向的工学博士，可以在智能检测算法的实现上为团队提出建设性意见；杨老师为服务外包大赛王牌指导教师，指导经验丰富，在往届服务外包大赛所带十余支队伍获得国家一、二等奖。两位老师在比赛全程各环节给予指导工作。本项目的设计使用了较新的技术和当前流行的结构框架，系统易用性和稳定性高、运行速度快。同时团队贯彻“用户至上，用心服务”的宗旨，操作流程和界面简单便捷。用户使用只需一部智能手机或一台装有基本浏览器的计算机，便可登陆系统，流程清楚明了。系统可以带来良好的使用体验，具有广泛而丰富的受众。

# 3.6法律可行性

法律可行性，是指法律与社会现实条件相适应，能够在社会中产生与立法意图一致的作用的程度或状况。法律与社会现实条件相适应的程度越高，产生与立法意图一致的作用越大，即可行性越大；反之，可行性越小。本项目完全遵守国家及国际发布的法律法规，遵照中国颁布的《中华人民共和国著作权》与《计算机软件保护条例》中的内容，项目产品创意均来自小组成员，绝无任何侵权行为，坚决支持正版，具有较高的法律可行性。
