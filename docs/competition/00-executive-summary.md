# Spectra 项目执行摘要

## 项目一句话定义

`Spectra` 是一套面向教学内容生产与课程资产建设的商业级知识空间系统。它把教师备课、资料进入、证据组织、内容生成、标准交付、正式沉淀和后续复用组织成同一条系统主线。

本项目的核心口号是：

**生产即沉淀，沉淀即交付。**

这句话对应系统设计原则：教师在正常生产教学内容的同时，系统自动形成可追踪、可复用、可演化的课程知识状态；每一次交付结果，也都可以成为后续课程空间继续增长的基础。

## 核心判断

教育数字化的关键问题，已经不只是“有没有工具生成内容”，而是：

- 教师的资料、旧课件、课堂经验和生成结果是否能被统一组织；
- AI 生成是否能被教师确认、修改和追踪，而不是一次性黑箱输出；
- `PPT`、教案、导图、动画、互动内容、说课稿和学情预演是否能成为同一知识空间的不同外化结果；
- 学校和机构是否能把教师个人成果沉淀为可治理、可复用、可持续演化的课程资产。

`Spectra` 的回答是：系统本体是库与引用关系。导出的 `PPTX`、`DOCX` 和多模态内容只是课程知识空间在不同场景下的按需外化结果。

更关键的是，这套判断已经进入现实系统：`Studio` 多模态内容工坊已经成立，`Session -> Diego -> Pagevra -> Ourograph` 主链已经成立，`Stratumind` 也已经在正式评估集上给出量化结果。`Spectra` 已经是一套开始成立的商业级系统。

## 产品形态

当前 `Spectra` 已经形成三个层次的产品形态：

| 层次 | 内容 | 价值 |
| --- | --- | --- |
| 课程知识空间 | `Project / Session / Artifact / Version / Reference / CandidateChange / Member` | 让课程内容具备长期状态、引用和演化能力 |
| `Studio` 多模态内容工坊 | `PPT`、`Word` 教案、导图、动画、互动小测、互动游戏、说课助手、学情预演 | 让一次备课派生多种可交付成果 |
| 六个正式能力层 | Diego、Pagevra、Ourograph、Dualweave、Stratumind、Limora | 让生成、渲染、知识状态、资料进入、检索和身份组织边界清晰 |

这六个能力层是项目围绕主链拆出的独立模块，不是简单罗列外部工具。它们会使用成熟组件、模型能力和公开方法，但最终服务于 `Spectra` 自己的课程知识空间、工作流契约和交付闭环。

这使 `Spectra` 交付成为一套围绕课程空间、知识回流、多模态外化和长期演化建立起来的教学知识系统。

## 已做成的关键能力

当前系统已经具备以下可说明、可展示、可继续演化的主体能力：

1. 以 `Project` 和 `Session` 组织一次完整备课过程，而不是只保留一条对话记录；
2. 支持多类课程资料进入统一链路，参与后续解析、检索和生成；
3. 支持基于 outline 确认、逐步生成、预览修改和标准导出的可管理生成主链；
4. 支持通过 `Studio` 承接 `PPT`、`Word` 教案、导图、动画、互动内容、说课稿和学情预演等外化结果；
5. 支持真实 preview / refine / history / download 约束，不把前端占位内容伪装成真实产物；
6. 支持将外化结果与正式知识状态绑定，沉淀为长期课程资产；
7. 支持围绕课程空间、版本、引用、候选变更和成员关系组织后续复用与演化。

## 关键证据

在检索增强这一最容易空泛化的能力上，系统已经具备正式评估集结果。

在 60 题正式集上，`Stratumind advanced` 相比 `dense_only`，Hit@3 从 `38.3%` 提升至 `80.8%`，Evidence Hit 从 `38.3%` 提升至 `80.8%`，二者均达到基础 dense 检索的 `2.11x`。

在 105 题正式集上，`Stratumind advanced` 相比 `dense_only`，MRR@3 从 `33.8%` 提升至 `68.6%`，Evidence MRR 从 `33.7%` 提升至 `67.6%`，均达到基础 dense 检索约 `2.0x`。

这说明系统已经在证据命中、证据排序和结果质量上形成了可量化优势。

## 商业价值

对教师而言，`Spectra` 减少资料整理、检索、排版和重复修改成本，让一次备课可以外化为多种教学成果。

对学生而言，课程内容不再只是一次性文件，而可以形成持续更新、结构更清晰、形态更多样的学习材料。

对学校和机构而言，`Spectra` 的价值在于把分散的教师个人成果转化为可治理、可复用、可持续演化的课程数据库。

更进一步看，`Project / 课程库` 可以成为可引用、可授权、可外化、可演化的课程知识资产节点。创作者、学习者、教研组和机构围绕课程库形成不同权限关系，平台则可以在工具订阅、课程资产授权、学习者增值服务、机构私有化和长期智能网络之间形成商业飞轮。

因此，本项目的商业价值不止于效率提升，而在于它具备从教师效率工具，走向课程知识资产平台，再走向 AI 原生个性化学习基础设施的自然路径。

## 执行摘要结论

`Spectra` 的真正竞争力，不在于单次演示中生成一份看起来不错的课件，而在于它已经把教学内容生产、知识状态管理、检索增强、标准交付、多模态外化和后续回流组织成了一套统一系统。

因此，本项目是一套面向真实教育场景、具备长期交付价值、平台生态潜力和持续演化能力的商业级课程知识空间解决方案。它建设的，是一层能够持续放大创作者价值、学习者价值和组织价值的长期教育内容底座。

从这个意义上说，`Spectra` 值得继续被认真评估，因为它已经把“生成、沉淀、交付、复用”从几件彼此分离的动作，组织成了一套开始成立的长期系统。
