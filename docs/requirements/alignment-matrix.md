## 详细对齐矩阵

### 功能 → AI 能力映射

| 功能编号 | 功能名称 | 依赖的 AI 能力 | 能力说明 |
|----------|----------|----------------|----------|
| **F01** | 项目创建与基础信息 | - | 纯业务逻辑，无 AI 依赖 |
| **F02** | 多轮对话需求采集 | C1 多模态意图解析 | 语义去模糊、主动追问、意图确认 |
| **F03** | 语音输入与术语纠错 | C1 多模态意图解析 | 语音识别 + 学科术语纠错 |
| **F04** | 多格式资料上传 | - | 纯业务逻辑，无 AI 依赖 |
| **F05** | 文档结构化解析 | C2 结构化重构 | PDF/Word 逻辑层级提取、公式识别 |
| **F06** | 视频关键帧提取 | C2 结构化重构 | 关键帧语义总结、时间轴分段 |
| **F07** | 本地知识库 RAG | C2 结构化重构 | 知识切片、向量化、语义检索 |
| **F08** | 教学意图融合引擎 | C1 + C2 + C3 | 意图+资料+RAG 融合生成指令集 |
| **F09** | PPT 课件生成 | C4 多端生成 | 文本→视觉映射、排版指令生成 |
| **F10** | Word 教案生成 | C4 多端生成 | 教学流程结构化输出 |
| **F11** | 预览与所见即改 | C5 增量优化 | 上下文关联记忆、差异化重构 |
| **F12** | 导出与交付 | C4 多端生成 | 多格式导出（PPTX/DOCX/HTML5） |
| **F13** | 来源溯源与引用 | C2 结构化重构 | 知识片段来源追溯 |
| **F14** | 教学法建议 | C3 教研规划 | BOPPPS/5E 教学法推荐 |
| **F15** | 学段学情适配 | C3 教研规划 | 难度自动调整、表达适配 |
| **F16** | 互动内容增强 | C4 多端生成 | 互动游戏逻辑脚本化 |
| **F17** | 历史版本管理 | C5 增量优化 | 版本快照、回滚能力 |
| **F18** | 模板库复用 | - | 纯业务逻辑 |
| **F19** | 分享与协作 | - | 纯业务逻辑 |

### AI 能力 → 技术实现映射

| AI 能力 | 能力描述 | 技术实现 | 相关 ADR |
|---------|----------|----------|----------|
| **C1** 多模态意图解析 | 语音/文字意图理解、主动追问 | Qwen 3.5 + Faster-Whisper + Prompt COT | ADR-004, ADR-008 |
| **C2** 结构化重构 | PDF/视频/图片 → 结构化知识 | MinerU + PaddleOCR + OpenCV + ChromaDB | ADR-005, ADR-006 |
| **C3** 教研规划 | 教学法推荐、学段适配 | Qwen 3.5 + 教学法知识库 + Prompt 模板 | ADR-004 |
| **C4** 多端生成 | PPT/Word/HTML5 生成 | Marp + Pandoc + Qwen 3.5 | ADR-007 |
| **C5** 增量优化 | 局部修改、版本管理 | Qwen 3.5 + 上下文快照 + Prompt 差异指令 | ADR-004, ADR-008 |
