#### 8.7 素材来源清单（精益求精 - 版权合规）

> 在教育行业，知识产权和素材溯源是刚需，体现产品成熟度

**触发方式**：
- 在下载界面勾选“素材来源清单（PDF）”
- 系统默认将清单添加到PPT最后一页，并可同时导出独立PDF

**清单内容（精简规范）**：
- 课件基本信息：课件名称、学段、总页数、生成时间、版本
- 来源统计：视频关键帧、文档引用段落、图片素材、AI生成内容占比
- 逐项追溯：来源文件名、定位信息（页码/时间戳）、用途说明
- 合规信息：版权声明、参考文献格式、侵权反馈联系方式

**用户配置项**：
- 清单格式：详细版 / 简化版 / 参考文献版
- 添加位置：PPT最后一页 / 独立PDF / Word附录（可选）
- 生成前支持预览与编辑，生成后支持二次导出

**智能提示**：
- 当检测到引用内容较多时，系统主动建议生成清单
- 当检测到潜在商业用途时，提示授权风险并推荐替代素材策略
