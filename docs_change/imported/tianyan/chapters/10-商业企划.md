# 9.商业企划

## 9.1基于PEST的市场环境分析

### 9.1.1 政策环境（P）

"互联网+交通"作为一个新兴行业，近年来受到国家的高度重视，为推动高新技术的研发，促进整体行业的发展，政府颁布了多项政策予以支持。工信部在2019年11月发布了《工业和信息化部关于加快培育共享制造新模式新业态促进制造业高质量发展的指导意见》，推动新型基础设施建设，加强5G、人工智能、工业互联网、物联网等新型基础设施建设，鼓励制造企业通过内网改造升级实现人、机、物互联，为共享制造提供信息网络支撑。

国务院印发的《交通强国建设纲要》、《国家综合立体网规划纲要》等文件对智慧交通的构建提出了进一步的要求。大量新城市区的出现带来了对智能交通系统的强烈需求，普遍采取的"一步到位"的建设方式催生了大批整体解决方案项目的出现。

### 9.1.2 经济环境（E）

近年来中国经济飞速发展，2021年中国GDP突破110万亿元，达114.4万亿元，较2020年增加了12.8万亿元，同比增长8.1%，两年平均增长5.1%。随着中国经济的高速发展，中国居民人均可支配收入也不断提升，人们已不仅限于对物质的需求，更加注重精神文化生活的追求，精神文化生活正成为人民日益增长的美好生活需要的重要组成部分，而交通行业作为为精神文化消费奠基的重要品种，必将迎来更大的发展空间。\
2021年，智能交通市场规模达1917亿元，同比增长15.62%，产业稳中向好。（预计2026年我国智能交通行业市场规模将突破4000亿元。）智能交通行业持续需求火热，资本利好智能交通领域，行业发展长期向好。下游行业交易规模增长，为智能交通行业提供新的发展动力。并且，2021年居民人均可支配收入35128元，同比增长9.1%，居民消费水平的提高为智能交通行业市场需求提供经济基础。

### 9.1.3社会环境（S）

传统智能交通行业市场门槛低，缺乏统一行业标准，服务过程没有专业的监管等问题影响行业发展。互联网与智能交通的结合，减少中间环节，为用户提供高性价比的服务。越来越多的大中城市管理者倾向于使用互联网＋智能交通作为城市交通拥堵，疏导交通流，获取交通数据的解决方案。并且，越来越多的90后、00后等人群，逐步接受智能交通系统，成为智能交通行业的消费主力和管理人员的重要部分。

当今人工智能领域已成为独角兽公司的集中地，在全球范围内，中国人工智能独角兽企业数量达到206家，领先其他国家；其次是美国，企业数量共有203家；然后是印度和英国，企业数量较少，分别为21家和13家。

### 9.1.4技术环境（T）

科技赋能人工智能、大数据、云计算、VR、5G 逐步从1、2线城市过渡到3、4线城市，实现智能交通行业科技体验的普及化。智能交通行业引入ERP、OA等系统，优化信息化管理施工环节，提高了行业效率。机器学习，深度学习相关研究的开展，也有效助力交通流疏导业务开展，能够用人工智能技术对车辆识别、轨迹甄别记录，流量统计等进行智能分析，让出行管理更高效，让通行更通畅。

### 9.2.3营销成本

在本系统开发成功后，为更好地提高项目的市场规模，发掘潜在消费群体，需开展营销。推广营销成本包括直接推销费用：直接销售人员的薪金、奖金、差旅费、训练费、交际费及其他相关费用预估为20000元。推广费用：广告媒体的成本、产品说明书的印刷费用、赠奖及展览会的费用、推广部门的薪金等，预估为40000元，总预估成本为60000元。

### 9.2.4风险成本

风险成本是指公司由于风险的存在和风险事故发生后人们所必须支出的费用和减少的预期经济利益。第一，风险损失的实际成本。风险损失的实际成本由风险造成的直接损失成本和间接损失成本共同构成。第二，风险损失的无形成本。风险损失的无形成本是指风险对社会经济福利、社会生产率、社会资源配置以及社会再生产等诸方面的破坏后果。第三，预防或控制风险损失的成本。为预防和控制风险损失，必须采取各种措施而支付的费用，预计30000元。

### 9.2.5学习成本

公司在进行项目开发前必须对所有参与研发的人员进行相关软硬件的培训，需要为他们购置必要的学习资料、相关计数设备等，预计15000元。

### 9.2.6售后成本

售后服务成本是指企业在产品销售后为消费者提供的一定服务而发生的费用。一般包括产品安装、调试费、顾客的培训费和介绍费，以及后续跟踪升级费用。本系统售后一定时期的维修费和损失费等，预计为50000元。

### 9.2.7其他成本

其他成本是指公司确认的除主营业务活动以外的其他日常经营活动所发生的支出。其他业务成本包括销售材料的成本、出租固定资产的折旧额、出租无形资产的摊销额、出租包装物的成本或摊销额等。采用成本模式计量投资性房地产的，其投资性房地产计提的折旧额或摊销额，也构成其他业务成本。企业应当设置"其他业务成本"科目，核算企业确认的除主营业务活动以外的其他日常经营活动所发生的支出，预计20000元，占比如图所示：

## 9.3成本控制体系

### 9.3.1确定战略目标

::: {custom-style="Normal (Web)"}
　　方向正确等于成功了一半，成本控制也一样。成本控制的目的是为了不断地降低成本，获取更大的利润，所以，制定目标成本时首先要考虑企业的赢利目标，同时又要考虑有竞争力的销售价格。由于成本形成于生产全过程，费用发生在每一个环节、每一件事情、每一项活动上，因此，要把目标成本层层分解到各个部门甚至个人。
:::

### 9.3.2企业项目分析

::: {custom-style="Normal (Web)"}
　　公司各个部门以营销目标导向，进行年度工作的项目立项，列出为实现目标所需要做的各类项目，同时对项目进行任务分解，再对时间、成本、性能每个环节进行分析，对比成本与收益。比如市场部明年为了达到既定的目标，需要完成多少市场宣传及推广的项目，项目逐一分解成任务后，对每个任务所需要的费用进行合理预算。同时对产生的收益进行估算。
:::

### 9.3.3价值链分析

::: {custom-style="Normal (Web)"}
行业价值链是：公司存在于智能交通领域行业价值链的某个点，如供应商产品的包装能减少公司的搬运费用，改善价值的纵向联系也可以使公司与其上下游和渠道企业共同降低成本，提高整体竞争优势。
:::

::: {custom-style="Normal (Web)"}
　　竞争对手的价值链和本公司价值链在行业价值链中处于平行位置，通过对竞争对手价值链的分析，可以测算出竞争对手的成本。然后，自己企业与之相比较，就找出了与竞争对手在任务活动上的差异，扬长避短，争取成本优势。
:::

### 9.3.4成本控制四步执行法

::: {custom-style="Normal (Web)"}
　　（1）减少目标不明确的项目和任务
:::

::: {custom-style="Normal (Web)"}
　　在企业目标清晰的情况下，每个项目及任务都是为实现目标所服务的。项目立项分析后，可以把目标不明确的项目与任务削减掉。
:::

::: {custom-style="Normal (Web)"}
　　（2）明确各部门的成本任务
:::

::: {custom-style="Normal (Web)"}
　　实行"全员成本管理"的方法。具体做法是先测算出各项费用的量高限额。然后横向分解落实到各部门，纵向分解落实到小组与个人，并与奖惩挂钩，使责、权、利统一，最终在整个企业内形成纵横交错的目标成本管理体系。
:::

::: {custom-style="Normal (Web)"}
　　（3）成本核算，精细化管理
:::

::: {custom-style="Normal (Web)"}
　　没有数字进行标准量化，就无从谈及节俭和控制。伴随着成本控制计划出台的是一份数字清单，包括可控费用(人事、水电、包装、耗材等)和不可控费用(固定资产折旧、原料采购、利息、销售费用等)。每月、每季度都由财务汇总后发到管理者的手中，超支和异常的数据就用红色特别标识。在月底的总结会议中，相关部门需要对超支的部分做出解释。
:::

::: {custom-style="Normal (Web)"}
　　为了让员工养成成本意识，建立《流程与成本控制SOP手册》。手册从原材料、电、水、印刷用品、劳保用品、电话、办公用品、设备和其他易耗品方面提出控制成本的方法。当然，有效的激励也是成本控制的好办法，所以，成本控制奖励也成为员工工资的一部分。
:::

::: {custom-style="Normal (Web)"}
　　（4）成本管理的提前和延伸
:::

::: {custom-style="Normal (Web)"}
　　将成本控制提前和延伸，提前就是加大技术投资，控制采购成本;延伸就是将上下游整合起来。
:::

## 9.5基于**SWOT**的分析

::: {custom-style="Normal (Web)"}
S（strengths）是优势、W（weaknesses）是劣势、O（opportunities）是机会、T（threats）是威胁。按照企业竞争战略的完整概念，战略应是一个企业"能够做的"（即组织的强项和弱项）和"可能做的"（即环境的机会和威胁）之间的有机组合。
:::

### 9.5.1优势**（S）**:

公司作为中小型企业，适应能力，灵活性高，可以灵活调整品种，结构和档次，而这正是公司相对于大型企业的优势所在。其次，科技创新能力强，科技成果转化运用快，和大企业的低效率高投入相比，公司的技术创新的驱动机制更为有效，创新要求更为迫切，创新成本更低下，创新成果转化更迅速。在此，发展空间更为广阔。同时，具有人力资本优势。当每一个人的聪明才智发挥才是决定效益和竞争力的主要因素时，小企业往往较比之大企业更具有优越性。

### 9.5.2劣势**（W）**：

在之后公司技术创新过程中，直接融资比较困难。在主板市场上市融资受到体制的制约，很难满足上市条件。同时受到自身条件限制和政策的制约。此外，在间接融资中，货款难，担保难。同时，知识产权保护体系不完备。目前我国知识产权保护由于在法律和管理体制等方面不够完善，存在着多头管理，侵权打击不力，侵权成本低。导致现阶段我国知识产权整体保护意识淡薄，保护水平低下，并且维权成本高，这是公司可能难以承受的。

### 9.5.3机会**（O）**：

时机优势。我国正处于政府机构改革的重要关头，中小科技企业是收纳失业人员，稳定社会的重要力量。另一方面，技术的进步促使一些产品的构造使得一些产品的构造变得极其精密复杂，大企业为了减少开发性投资和保持自身经营的灵活性，越来越需要本公司参与其配套产品的制作，因此，从中央到地方，从专家学者到政府官员都给予了中小企业极大地关注和支持，人们传统观念上对中小企业的歧视己经不复存在。

### 9.5.4**威胁（**T**）**：

公司本身具备的竞争优势相对于大型创新企业仍有相当大的距离。不具备过硬的竞争实力和品牌能力，难以与大中型企业抗衡。无独有偶，近些年来我国为了完善市场经济不断调整市场准入和市场退出的限制，国际环境的变化，更是使得中小企业走出去的发展目标更加遥遥无期。近年来随着我国人口红利的渐渐消失，人力资源成本不断提升，加之熟练工人流动性的增加，企业工人的薪资、五险金、培训新人等成本的不断增加，另一方面又不断地压缩企业的利润空间，更造成企业难以开工。由于公司在管理、融资渠道、核心技术资本、人才和品牌等转型升级的重要因素方面积累不够，所以在转型升级上很困难。

## 9.6商业模式

### 9.6.1客户细分

主要用户：全国各地交通管理单位，部门以及相关企业

### 9.6.2盈利模式

盈利模式是企业在市场竞争中逐步形成的企业特有的赖以盈利的商务结构及对应的业务结构。

**（1）项目盈利的商务结构**

商务结构反映的是企业内部资源整合的对象及其目的。企业的商务结构主要指企业外部所选择的交易对象、交易内容、交易规模、交易方式、交易渠道、交易环境、交易对手等商务内容及其时空结构。

本项目的交易对象为各中大城市交通部门，通过销售项目产品实现盈利。我们将为合作渠道提供产品及整体解决方案（技术方案）、宣传材料、项目实施与售后。在交通拥堵已成为大中城市交通中的普遍现象背景下，本项目产品能够较好地降低大车流、大事故、大拥堵带来的肇事风险，保障城市交通安全、确保道路畅通，具有极好的市场交易环境。

运营层主要负责素材管理、采购及仓储以及产品研发和营销。素材管理包含产品介绍、案例介绍、使用介绍、活动库等，向客户充分介绍项目产品。采购及仓储负责进行产品的实体制作。产品研发能够提高产品的市场竞争力，提高项目的利润杠杆和利润屏障。产品营销负责通过内外部渠道，吸引客户购买和使用企业产品或服务。

产品层主要包括基础产品、三方产品和创新产品。在基础产品之上，三方产品能够补全产品生态，让本项目产品使用用户粘性更高。创新产品能够满足用户群体的更多需求，通过具体场景，寻找更多的机会点，争取将创新产品打造成基础产品。

用户层负责监管项目用户群体（利润源），主要为项目商品、服务的购买者和使用群体。在服务运营过程中，通过针对客户清晰的偏好，为客户创造价值，间接地为企业创造价值。同时挖掘潜在的客户，扩大已有客户规模，增加竞争优势。

**（3）项目盈利模式的类型**

盈利模式分为自发的盈利模式和自觉的盈利模式两种。

本项目主要希望以自觉的盈利模式进行运营盈利。通过前期的预盈利模式进行运营探索，前中期通过对盈利实践的总结，针对盈利模式进行自觉地调整和设计，针对市场竞争不断优化改进。这样形成的盈利模式具有清晰性、针对性、相对稳定性、环境适应性和灵活性的特征。

本项目盈利模式更好地规避了自发盈利模式企业对如何盈利，未来能否盈利认识不清晰，或者企业虽然盈利，但盈利模式不明确不清晰，盈利模式具有隐蔽性、模糊性、缺乏灵活性的缺点。

**（4）项目盈利模式特点**

本项目通过销售产品和服务实现盈利。如今行业宏观和微观经济环境都处于不断变化的状态中，没有一个单一的特定盈利模式能够保证在各种条件下都产生优异的财务结果，一个较好的盈利模式一定是能够根据市场形势不断做出调整且具有相应特色的。本项目的盈利模式主要具有以下特点：

1.产品和服务能够提供独特价值。除了项目产品基础的作用外，我们还设有技术研发部门对产品和服务的独特性组合进行研究，向客户提供额外的价值，甚至使得客户能用更低的价格获得同样的利益，或者用同样的价格获得更多的利益。

2.难以模仿的盈利模式。本项目通过确立自己的与众不同，比如对客户的悉心照顾、独特的设计理念、强大的研发能力等，来建立利润屏障，提高行业的进入门槛，从而保证利润来源不受侵犯。

3.脚踏实地的盈利模式。脚踏实地就是实事求是，就是把盈利模式建立在对客户行为的准确理解和假定上。项目企业做到量入为出、收支平衡。年复一年、日复一日脚踏实地地做到完成。做到对于自己的钱从何处赚来，为什么客户看中自己企业的产品和服务，有一个清晰的了解和认识。这样才能更好的为企业带来利润，不至于甚至在无意识中侵蚀企业的收入。

**（5）盈利策略**

产品策略：从实际出发，服务客户；由点到面，由核心到外围；树立口碑，留住客户，打造自身品牌。在系统基本主体确定情况下，采用"量体裁衣，因地制宜"的产品策略，根据不同交通管理单位的实际情况与要求，按预留接口对系统做修改，为不同的交通部门带来不同的体验，使其都收获到满意的用户体验，巩固客户。

价格策略：采用统一定价原则，并且可以略低于市场价格。这样可以保证销售的公平性，赢得品牌口碑。

广告策略：广告营销对于购买意愿有显著正向影响。公司不仅要抓住电视广告途径，更抓住现阶段短视频红利时期，加大在线上平台的宣传推广力度。

### 9.6.3线下业务营销策略

策略一：编写科学的产品销售手册

雇佣一批精干的销售力量，认真研究竞争对手产品以及销售经验，设计一系列产品销售技巧及产品特点演示方法，为用户购买智能交通导流系统带来科学的理由。

策略二：点对点推销

公司派出优秀的营销团队，到各交通管理单位等实施点对点推销，与公司管理层、普通员工等分别进行互动交流。这种点对点的方法直接有效，针对性强，信赖度高，在初期推广中宜大范围使用。

### 9.6.4线上业务营销策略

线上营销按通俗的意义来讲就是以互联网线上为基础开展的推广活动，任何以这个推广活动为目的所用到的渲染手段都属于线上营销。这种方式正逐渐变得不可或缺，成为每个品牌传播的必经之道。其主要特点是成本低，效率高，传播广，效果好，及时性。

**策略一：打造品牌形象**

公司通过新闻媒体采访公司人员以及相关用户，得到用户使用本公司智能交通产品的体验，创建百度百科，塑造线上的品牌形象，可以提高公司智能交通产品的公信力。

**策略二：树立品牌口碑**

鉴于百度贴吧的高人气和百度作为中国最大的搜索引擎的特殊性，百度贴吧依然有一定的使用价值。并且，豆瓣也是开展论坛营销的主要平台之一。

**策略三：引流客户购买**

通过知乎，抖音，小红书，b站，简书等自媒体平台对公司智能交通系统进行线上推广。同时，微信、新浪微博等平台是企业开展线上营销都需关注的重点平台，能为企业高效带来客户。

### 9.6.5企业形象推广策略

**策略一：人物形象宣传**

在有别于个人树碑立传式的公司形象建设中，应把公司领导层个人宣传和企业的战略目标、公关目标结合起来以达到好的宣传效果。可以建立、开发可提高企业管理者声誉的平台来实现宣传效果。公司可将蕴含高层管理者有代表性的战略思想和管理策略的文章提供给媒体发表；定期或不定期举办专家座谈会、研讨会和媒体座谈会，建立良好的声誉信息等。

**策略二：文化形象宣传**

企业成功的形象宣传并非只靠华而不实的广告来打响知名度，这一切都必须将企业本身所代表的意念和文化深植于客户的观念之中，才能藉以提升客户对企业产品的忠诚度，在不过多挪用公司紧缺资金的同时达到宣传企业形象的目的。而企业文化是作为一个观念系统存在的，要传达到企业员工、社会和公众存在一些困难。因此，将企业文化特质形成一个统一概念，通过个性化、鲜明的视觉形象(图形、图案)表达出来，再传导给社会。
