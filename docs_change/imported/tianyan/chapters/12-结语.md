# 11.结语

本项目使用物联网+视觉分析和深度学习等技术实现单向行驶多车道智能识别车牌、监控车辆与行人流量信息、追踪特殊车辆及对违法车辆报警提示等功能。结合实际路口交通控制逻辑，通过Optimal timing智能算法，实现可及时调控车流及人流的智能交通信号灯。同时，搭建智能交通大数据监测分析平台，完成数据监控可视化和对后台指挥控制系统的管理；并且能够使小屏控制大屏，实现多屏协同工作。

在当今社会，伴随着人工智能行业的飞速发展，中国的社会经济发展进入全新阶段，同时也为交通强国的建设带来了新的战略机遇。本项目的成功落地实现，积极相应了《数字交通发展规划纲要》、《推动中国交通大数据发展行动纲要》政策号召，贯彻了建设交通强国时代要求，是当代青年大学生立足国情、着眼全局、面向未来做出的不懈努力，是深度落实"十四五"加快建设交通强国的政策方针的坚实步伐！
