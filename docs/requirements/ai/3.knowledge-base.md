

### 3. 知识库规划（RAG 架构）

#### 3.1 知识库内容维度规划

为了让 Spectra 具备专业“教研员”的素质，知识库分为四个核心层级：

1. **核心学科层 (Domain Knowledge Layer)**：
   
   * **标准教材**：经 MinerU 解析后的 Markdown 格式教材（保留 LaTeX 公式、表格、目录结构）。
   * **教学大纲**：国家或校方颁布的课程标准、教学目标、重难点分析。
   * **题库与案例**：配套习题、经典教学案例、各学科专有名词词典。

2. **多模态素材层 (Multimodal Asset Layer)**：
   
   * **视频指纹库**：由 Faster-Whisper 生成的带时间戳的语音文本 + OpenCV 抽取的关键帧图像描述（VLM 生成）。
   * **图表索引**：教材插图、实验装置图的 OCR 识别内容及语义标签。

3. **教研方法层 (Pedagogical Logic Layer)**：
   
   * **教学模型库**：存储 BOPPPS、5E 教学法、翻转课堂等标准流程的逻辑模板。
   * **互动策略库**：常见互动小游戏的规则脚本、课堂提问技巧。

4. **视觉资产层 (Visual Layout Layer)**：
   
   * **PPT 组件库**：预设的 PPT 排版样式（JSON 描述格式）、配色方案、图标资产。

---

#### 3.2 知识组织与切片策略 (Advanced Chunking)

在自研方案中，切片（Chunking）的质量直接决定了 AI 生成课件的逻辑性。

* **分级切片法 (Recursive Character Splitting)**：
  * **一级切片（章/节）**：用于宏观教学大纲的规划。
  * **二级切片（知识点/概念）**：300-500 Token，作为 RAG 检索的主体。
  * **三级切片（原子要素）**：如一个公式、一张图、一个例题，用于填充 PPT 页面。
* **父子文档索引 (Parent-Child Retriever)**：
  * 在向量数据库中，子文档（小块）用于计算向量相似度以提高检索精度，而一旦命中，系统会通过关联的 ID 召回父文档（大上下文），确保 Qwen/GPT 生成的内容具备连贯性，不至于断章取义。

---

#### 3.3 存储与检索机制设计

1. **混合存储引擎 (Hybrid Storage)**：
   
   * **向量数据库 (ChromaDB / Milvus)**：存储文本和图像的 Embedding（嵌入向量），实现语义搜索。
   * **关系型数据库 (PostgreSQL / SQLite)**：存储元数据（如：所属章节、页码、视频时间戳、学科分类）。
   * **可考虑：对象存储 (MinIO / 阿里云OSS)**：存储原始的图片、PDF 附件。

2. **检索流水线 (Retrieval Pipeline)**：
   
   * **Step 1：混合搜索 (Hybrid Search)**：结合 **BM25 算法**（针对“牛顿第一定律”等专有名词的精确匹配）与 **Vector Search**（针对语义模糊匹配）。
   * **Step 2：知识重排序 (Rerank)**：使用本地部署的 **BGE-Reranker** 模型，对初次召回的 Top-20 结果进行深度打分，挑选最匹配的 3-5 条送入大模型。
   * **Step 3：多模态对齐检索**：当老师搜索“光合作用视频”时，不仅检索视频文件名，还检索 Faster-Whisper 转录出的文本内容，实现**“搜词找帧”**。

---

#### 3.4 知识库的动态维护与溯源

* **“棱镜”溯源功能 (Knowledge Citation)**：
  * 要求自研系统在生成 PPT 时，必须在元数据中标记该知识点来源于哪份 PDF、第几页或视频的第几分钟。
  * **用户价值**：老师在预览课件时，点击“查看原文”即可立刻弹出教材对应位置，极大增强系统可信度。
* **反馈闭环 (Feedback Loop)**：
  * 当老师手动修改了某页 PPT 的知识点表述，系统应提供“存入知识库”选项，通过增量 Embedding 完善个人/学科专属知识库。

---

### 核心设计亮点：

知识库通过 **Parent-Child 递归索引** 解决了传统 RAG 无法处理教学逻辑连贯性的痛点。结合 **MinerU 的排版感知解析**，Spectra 能够精准区分教材中的‘定义’与‘例题’，在检索阶段即可根据 PPT 页面类型（是讲定义还是讲例题）动态过滤知识块，实现了真正的**场景化检索**。
