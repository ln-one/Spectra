# 10.风险管理

## 10.1风险识别

风险识别是风险管理的首要环节，本项目小组将完成本项目，希望通过分析发生风险的原因，提前对风险进行识别，在整个项目完成过程中尽力降低各种风险带来的项目损失。我们将风险分为以下五个方面。

### 10.1.1法律风险

本项目的研究，均是在小组成员自行学习和老师指导下完成，对系统收集的数据及图像不进行任何非法传播和使用。团队对作品、技术方案以及外观设计拥有合法权利，并且保证全部作品及资料的任何部分均不侵犯任何第三方的知识产权或专有权利，不含任何诽谤、淫秽或非法材料。系统的开发及设计全部遵循国家有关法律法规，项目符合规范，不存在侵权等现象。

### 10.1.2经营风险

本项目成员均为在校大学生，在实际的经营工作方面缺少经验，生产运营经营过程还需要多学习实践。团队对于经营决策引起的未来收益具有不确定性。现今已有的组织架构和管理机制伸缩性较强，可以根据需求进行及时调整，但其是否适合长期的可持续发展还需要观察。做出的决策仍需要在未来项目进程中不断践行实践和调整。同时，由于社会结构变动，如地震、经济危机等带来的市场环境变动不可预知，这些可能对该项目带来一定风险。

### 10.1.3管理风险

在整个项目的管理运作过程中，本项目组成员来自多个学院，专业知识全面。在组长的带领下自学python、深度学习、web搭建等相关知识，具有较强的专业素养和团队协作能力，对本项目的完成有很大信心。同时团队组织架构合理，分工范围明确，具有较强的团队凝聚力和丰富的团队经验。在项目完成过程中，项目组成员尽力获取最新信息，避免由于信息不对称带来判断失误。

### 10.1.4财务风险

在项目完成过程中，开发的支出仅为项目开发需要的基础设备，耗材使用、购买参考书等学习资料，一些简单的硬件购买成本以及少量市场开拓、广告、运营，后期维护成本。而本项目完成度高，系统性，一体化强，对实地路口的交通流量识别和预测有高精确度的优点；能够根据当前流量实现信号灯的智能控制导流，同时设计了APP与web页面等多种控制方式，方便使用人员进行操作。若投入到大中城市交通部门中使用，所收获的效益远大于成本，使企业蒙受经济损失的可能性较低。

### 10.1.5市场风险

国务院颁布的《交通建设强国纲要》指出，我国将大力发展智慧交通，在当下大车流、大事故、大拥堵已经成为大中城市普遍现象，各城市对智能信号灯系统改进、数据汇总、自动识别、非法监控、屏幕交互提出了更大的需求和要求。如今信号灯行业地位逐步提高，智能信号灯市场覆盖范围规模不断增大，服务用户种类提高。智能信号灯系统发展前景一片向好。然而现在正处于科技高速发展的时期，技术变革和行业革新对信号灯行业市场会带来较大的影响，对我们的技术竞争力提出更高的要求，这些可能对该项目带来一定风险。

## 10.3风险控制及效果评价

该项目组主要采用了风险回避的方式进行风险控制。在项目开展初期，各项目组成员对该项目进行了充分调查及数据收集，通过网络资料查找，实地考察等多种方式，最大程度上规避信息不对称的风险。同时还建立了项目组风险防范体系，进行法律、经营、管理、财务、市场风险的控制，有效防范风险，保障项目组系统经营安全。

本项目组的风险管理体系如下：

风险识别和统计：项目组在初期进行初次风险识别，各成员定期汇报项目进展，及时更新下一阶段项目设计。在每周例会中汇报和统计项目现阶段可能存在的风险，更新使用数据，讨论提出风险控制方法。

风险监控和控制：对已提出风险的一些关键风险指标和环节进行定时监测，建立风险预警机制，观察所采取的风险管控措施质量和效果，规避项目发展方向出现问题，在风险监测的过程中不断积累经验，共同确保风险控制稳定实施。

该项目组在项目开展过程中，各项目组成员按照风险管理体系认真履行各自的职责，做好风险控效果评价工作，为预防和控制事故提供了可靠依据。在整个过程中，风险识别较全面，风险控制措施充分有效，对控制过程中产生的新风险及时识别控制，采取措施消减风险，将风险控制在可以接受的程度，无重大风险发生，达到了预防事故，确保项目安全进行的目的。
