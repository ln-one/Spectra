## 关键设计

### 系统边界
- **In Scope**: 项目管理、需求采集、资料处理、RAG、课件生成、预览修改
- **Out of Scope**: 教务系统、实时协作、课堂执行、内容担保、版权授权

### MVP 闭环
```
项目创建 → 对话输入 → 资料上传 → RAG召回 → 意图融合 → 课件生成 → 预览修改 → 导出
```
