# 成员 C 给成员 D 的协作交接（2026-03-13）

## 1）Prompt 接线入口（/chat/messages）

后端真实入口：
- `backend/routers/chat.py::send_message`

Prompt 拼装实际链路：
1. 先根据用户输入与系统提示拼装 `user_message_for_prompt`。
2. 将检索结果整理为 `rag_payload`。
3. 调用 `prompt_service.build_chat_response_prompt(...)` 生成主 Prompt。
4. 在最前面追加项目上下文：`项目：{project.name}`。
5. 调用 `ai_service.generate(...)`，并携带 `route_task=chat_response`。

结论：
- 成员 D 在 `prompt_service` 里的 prompt/few-shot 改动，会进入线上真实调用链，不会只停留在模板层。
