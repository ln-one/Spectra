# 3可行性分析

## 3.1可行性分析

### 3.1.1社会可行性

该系统严格符合道德标准，未涉及对人民隐私相关内容的侵犯。并且，符合广大人民群众的热切需求，着力解决当下百姓交通"难"，出行"堵"问题，推动交通运输供给加快从"走得了"向"走得好"转变。努力做到让广大人民群众享有更便捷的交通运输，获得更加公平、更有效率的交通服务，不断增强人民群众的获得感、幸福感、安全感，为公司带来广泛社会影响力。

### 3.1.2政策可行性

交通强国是实现第二个百年奋斗目标的重要支撑，交通现代化是国家现代化的重要标志。同时，国家部署了智能制造等国家重点研发计划重点专项，印发实施了"互联网+"人工智能三年行动实施方案，从科技研发、应用推广和产业发展等方面提出了一系列利好措施。

随着人民出行和货物运输需求的深刻变革，客货物运输的可达性和时效性要求更高，亟待提升运输服务的效率和品质。新一轮科技革命和产业变革加速演变，智慧交通、绿色交通、共享交通成为各国培育交通发展新优势的重要发力点。国家推进特种装备研发应用，推广智能化、数字化、轻量化交通成套技术设备；推动发展智慧交通，推动大数据等新技术与交通行业深度融合，用新技术为传统交通基础设施赋能，争当新基建主力军。

### 3.1.3技术可行性

本组成员都有一定的组织编程开发项目经验，来自多个学院，专业知识全面，自学python，深度学习，web搭建相关知识，对本项目的完成有很大信心，具有较强的自学能力和团队协作能力。项目使用python作为语言基础，使用灵活，方便，拓展性强。深度学习的相关技术也是项目实现的有力保障。虽然项目工作量较大，一些技术开发存在技术难点，但通过开发用户的技术攻关和良好的协调安排，小组成员有能力在比赛限定时间内完成项目。因此本项目在技术上可行。

### 3.1.4经济可行性

完成该智能系统的开发的支出仅为项目开发需要的基础设备，耗材使用、购买参考书等学习资料，一些简单的硬件购买成本以及少量市场开拓、广告、运营，后期维护成本。而本项目完成度高，系统性，一体化强，对实地路口的交通流量识别和预测有高精确度的优点；能够根据当前流量实现信号灯的智能控制导流，同时设计了APP与web页面等多种控制方式，方便使用人员进行操作。若投入到大中城市交通部门中使用，所收获的效益远大于成本，因此经济可行。

### 3.1.5实践可行性

小组有两名指导教师，李老师为交通控制方向的工学博士，可以在深度学习，智能交通导流算法的实现上为团队提出建设性意见；杨老师为服务外包大赛王牌指导教师，指导经验丰富，在往届服务外包大赛所带十余支队伍获得国家一，二等奖。两位老师在比赛全程各环节给予指导工作。

本项目的设计使用了较新的技术和当前流行的结构框架，系统易用性和稳定性高、运行速度快。同时团队贯彻"用户至上，用心服务"的宗旨，操作流程和界面简单便捷。用户使用只需一部智能手机或一台装有基本地浏览器地计算机，便可登陆系统，流程清楚明了。系统可以带来良好的使用体验，具有广泛而丰富的受众。
