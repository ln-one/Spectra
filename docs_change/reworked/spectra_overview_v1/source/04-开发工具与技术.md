# 开发工具与技术

前端采用 `Next.js + React + TypeScript`，后端采用 `FastAPI`，数据底座由 `PostgreSQL`、`Redis`、`RQ`、`Qdrant` 等组件组成。技术路线重点围绕会话生成、知识库检索和标准导出展开。
