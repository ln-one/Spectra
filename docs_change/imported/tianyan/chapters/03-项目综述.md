# 2.项目综述

## 2.1项目背景及意义

在交通拥堵已成为大中城市交通中的普遍现象背景下，交通形势严峻，大车流、大事故、大拥堵带来的肇事风险、交通应急风险等隐患突出，城市交通保安全、保畅通压力大，每年因交通事故引起的直接损失折款多达几百亿。根据最新的交通拥堵情况报告，北京、西安、上海、青岛、成都通勤高峰拥堵指数分别高达2.063、1.987、1.932、1.838、1.763。交通拥堵不但浪费民众大量的时间和精力，增加生产生活成本。同时，堵车还会带来环境污染问题，在怠速状态下，汽车尾气排放量是正常行驶的20―30倍。而国内智能交通管理系统的缺位，相关方案难以普及实施，正是造成这一现象的重要因素。

本项目响应《国家综合立体交通网规划纲要》要求：坚持创新核心地位，注重科技赋能，促进交通运输提效能、扩功能、增动能。推进交通基础设施数字化、网联化，提升交通运输智慧发展水平。本项目的开发和实现，对于促进城市交通减少拥堵，疏导城市高峰交通车流人流，构建便捷顺畅、经济高效、绿色集约、智能先进、安全可靠的现代化高质量国家综合立体交通网，加快建设交通强国有重大意义。

## 2.2项目目标

贯彻落实中共中央，国务院《交通强国建设纲要》、《国家综合立体网规划纲要》等文件精神，创新建设智慧交通平台，为我国早日建设智慧型交通强国打下坚实的基础。

实现依据人流车流的智能交通信号灯疏导算法控制信号灯，减少道路拥堵，从而实现绿色低碳交通，让通行更通畅；通过视觉分析，深度学习等智能技术实现多车道车辆识别、轨迹甄别记录，实现车流量，人流量统计等进行智能分析，搭建可视化智能交通大数据平台，所有数据实现可视化大屏呈现，打造"一站式"智慧交通系统。

## 2.3项目亮点

（1）高完成度：项目完成度高，精准聚焦各项任务要点难点，逐一突破，较好实现了项目预期的全部目标。并且，项目在已有要求的基础上增加了APP操控，VISSIM仿真等额外内容。

（2）高优化度：综合了卷积层神经网络模型和贪婪算法对历史，当下，未来车流进行统计建模，不断迭代。创新的提出优化交通灯配时的交通流疏导算法------Optimal timing算法。经过matlab仿真得到该算法相较于固定控制算法和模糊评价算法有显著的优越性。

（3）高精确度：项目提出的ridge regression岭回归预测未来车流进入迭代中，车流预测准确性达93%。

（4）高可行性：项目积极响应"十四五"规划加快建设交通强国政策，将成果采用软件+硬件的呈现形式，给出从车流检测到交通疏导的一站式方案，为交通强国献上青年方案。
