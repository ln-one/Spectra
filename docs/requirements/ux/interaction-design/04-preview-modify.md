**教案同步视图功能**：

1. **自动同步**：

   - 切换PPT页面时，教案视图自动跳转到对应页面的教案
   - 实时联动，无需手动切换

2. **教案内容结构**：

   ```
   每页教案包含：
   • 🎯 教学目标（本页）
   • 📝 教师话术（建议讲解内容）
   • 💡 教学建议（互动方式、提问技巧）
   • ⏱️ 建议时长
   • 📎 素材来源（溯源信息）
   • 🎓 学生活动（预期反应）
   • ⚠️ 注意事项（易错点、重难点）
   ```

#### 6.5 修改方式

**方式1：对话式修改**（推荐）

- 用户："第3页的图片太小了"
- AI：理解并调整，显示新版本
- 用户："第5页的例题太难，换一个简单的"
- AI：生成新例题

**方式2：直接点击修改**

- 点击PPT元素
- 对话框自动输入："我要修改第3页的..."
- 用户补充具体要求

**方式3：快速操作按钮**

- 调整顺序：拖拽缩略图
- 增加页面：选择位置和类型
- 删除页面：点击删除按钮
- 修改风格：选择预设风格
