# 8-2 商业飞轮图

## 版本

`答辩版`

## 适配场景

`PPT 横向`

## 图类型

`闭环 / 商业图`

## 这张图只回答什么

业务增长如何形成“内容沉淀 -> 资产复用 -> 效果提升 -> 用户增长”的飞轮。

## 主阅读路径

沿环形或横向闭环看六步，再回到起点。

## 来源与事实锚点

- `docs/competition/08-business-plan.md`
- `docs/competition/08-business-plan-src/02-flywheel.md`

## 现有图问题检测

- 原骨架可用
- 需要更清楚的中文短词和布局约束
- `结论`：`保留骨架，小修`

## 信息分层设计

- 飞轮步骤层
- 中心价值层

## 分组设计

- 中心：平台价值
- 外环：六步飞轮

## 密度策略

- `中密度`
- 答辩版允许有视觉压强，但标签必须更短

## 画幅与布局约束

- `16:9` 横向
- 环形或横向回环
- 每步只用短词

## 优化后的 Mermaid 骨架

```mermaid
flowchart LR
    A["内容沉淀"] --> B["资产复用"] --> C["交付提效"] --> D["效果提升"] --> E["用户增长"] --> F["更多数据"] --> A
```

## 中文手绘主 Prompt

请重绘一张用于中国高校竞赛答辩 PPT 的商业飞轮图。  
这张图是 `16:9` 横向图。  
飞轮可以做成环形，也可以做成横向回环，但必须一眼能看懂六步：`内容沉淀`、`资产复用`、`交付提效`、`效果提升`、`用户增长`、`更多数据`。  
中心可以有一个简洁的 `平台价值` 节点。  
整体风格专业、高级、低饱和、克制、简约多彩，适合答辩展示。  
这张图可以有一点视觉压强，但文字必须短，不能挤。

## 英文补充关键词（可选）

## 统一风格负面约束

- 禁止每步写成长句
- 禁止太多箭头叠在一起
- 禁止高饱和营销海报风
- 禁止小字

## 审图备注

- 飞轮感要强。
- 六步标签必须短而稳。
